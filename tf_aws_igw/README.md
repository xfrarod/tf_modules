Internet Gateway Terraform module
=================================

A Terraform module to provide an Internet Gateway in AWS.


Module Input Variables
----------------------

- `vpc_id` - VPC id
- `name` - Name (optional)

Usage
-----

```js
module "igw" {
  source = "github.com/terraform-community-modules/tf_aws_igw"

  name   = "default"
  vpc_id = "vpc-12345678"
}
```

Outputs
=======

 - `igw_id` - IGW id
